\freesection{Заключение}

Основными итогами можно считать достижение всех целей курсовой работы:
\begin{itemize}
\item получен навык решения практических задач;
\item изучена методика подготовки, проведения анкетирования и обработки результатов;
\item использована техника иерархического синтеза для принятия сложных решений при наличии многих альтернатив;
\item улучшены навыки алгоритмического мышления.
\end{itemize}

Кроме этого, выполнены все задачи курсовой работы:
\begin{itemize}
\item решены предлагаемые практические задачи;
\item проведены все этапы иерархического синтеза;
\item подготовлена пояснительную записку.
\end{itemize}
