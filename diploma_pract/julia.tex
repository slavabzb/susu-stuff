\subsection{Язык программирования Julia}
\label{julia}

Язык программирования Julia -- это высокоуровневый высокопроизводительный динамический язык программирования для технических вычислений\cite{Balbaert2016}. Синтаксис языка очень похож на синтаксис других популярных сред для технических расчетов. В распоряжении имеются умный компилятор, распределенное параллельное исполнение, численная точность и большая библиотека математических функций. Базовая библиотека Julia, в большинстве своем написанная на Julia, также содержит лучшие открытые библиотеки C и Fortran для линейной алгебры, генерации случайных чисел, обработки сигналов и работы со строками. Общество разработчиков Julia поставляют большое количество внешних пакетов через встроенный пакетный менеджер. По аналогии с популярной утилитой Jupyter, существует многофункциональный браузерный графический интерфейс IJulia.

Ниже перечислены некоторые из встроенных возможностей Julia\footnote{Подробнее на http://julialang.org.}.

\begin{itemize}
\item Мультидиспетчеризация: возможность определить поведение функции, основываясь на различных комбинациях входных параметров.
\item Динамическая система типов: типы для документации, оптимизации и диспатчинга.
\item Хорошая производительность, сравнимая со статичиски компилируемыми языками такими, как C.
\item Встроенный менеджер пакетов.
\item Макросы, подобные макросам из Lisp, и другие возможности метапрограммирования.
\item Вызов функций Python.
\item Вызов функций C напрямую, без применения оберток или специального API.
\item Широкие возможности для управления другими процессами.
\item Архитектура, специально спроектированная для параллельных и распределенных вычислений.
\item Легковесные "зеленые" потоки.
\item Объявление пользовательских типов, таких же быстрых и компактных, как встроенные.
\item Автоматическое создание эффективного специализированного кода для различных типов аргументов.
\item Элегантное и расширяемое преобразование численных и других типов.
\item Поддержка Unicode, включая (но не ограничиваясь) UTF-8.
\item Лицензия MIT: бесплатность и открытость исходного кода.
\end{itemize}

Основанный на LLVM JIT-компилятор Julia и дизайн самого языка позволяют ему по производительности приблизиться и часто сравняться с производительностью языка C\cite{Kwon2016}.

Julia не заставляет пользователя следовать какому-либо определенному стилю параллелизма\cite{Tate2014}. Вместо этого, в распоряжении имеется множество ключевых строительных блоков для распределенных вычислений, которые позволяют использовать различные стили и добавлять новые.

JuliaBox позволяет запускать ноутбуки IJulia в контейнерах-песочницах Docker. Это открывает простор для полностью облачной обработки, включая управление данными, редактирование и обмен кодом, выполнение, отладки, анализа и визуализации\cite{Rohit2016}. Цель этого проста -- перестать беспокоиться об администрировании машин и управлении данными и сразу перейти к решению задачи.
