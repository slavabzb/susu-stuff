\freesection{Заключение}

В ходе проведенной работы:
\begin{itemize}
\item углубились знания в области дискретной математики,
\item приобретен навык проведения практических занятий,
\item приобретен опыт индивидуального консультирования по сложным вопросам предмета,
\end{itemize}

Очень важным, если не определяющим фактором, способствующим успешному прохождению педагогической практики, явилось доброжелательное, участвующее отношение преподавателей: оказывалась всякая помощь, давались ценные советы по разработке и проведению занятий. В свою очередь, студенты проявляли дисциплинорованность, одобрение и заинтересованность, демонстрировали хорошую посещаемость в семестре и успеваемость на экзамене.

В ходе практики удалось реализовать все поставленные цели и задачи:
\begin{itemize}
\item приобрести бесценный практический опыт и навыки работы с коллективом студентов с учетом его психологической структуры и уровня развития;
\item углубить свои знания в дискретной математике и педагогике;
\item сформировать умения по организации продуктивного взаимодействия с группой на паре (установление личных контактов, навыки сотрудничества, диалогового общения и т.д.),
\item развить умения выявлять, анализировать и учитывать при организации учебно-воспительного процесса общие психологические закономерности;
\item умение помечать и анализировать возникающие в коллективе ситуации, требующие педагогического вмешательства.
\end{itemize}


В работе представлена параллельная реализация алгоритма метода эллипсоидов.

В работе решены следующие \textbf{задачи}:
\begin{itemize}
\item операции классического алгоритма метода эллипсоидов исследованы на вычислительную сложность;
\item разработана программная реализация алгоритма с распараллеливанием наиболее длительных по времени операций;
\item обеспечена поддержка арифметики расширенной и произвольной точности;
\item продемонстрировано использование разработанного ПО для решения задачи оптимизации большой размерности;
\item разработанный код проверен и протестирован.
\end{itemize}

На основе анализа результатов вычислительных экспериментов можно сделать следующие выводы:
\begin{itemize}
\item параллельная реализация метода эллипсоидов позволяет за то же время решать задачи оптимизации большей размерности;
\item разработанное ПО, поддерживающее параллельное выполнение операций, эффективнее использует ресурсы современной вычислительной системы;
\item для задач оптимизации большой размерности разработанная реализация выполняется быстрее однопоточного алгоритма и имеет ускорение, превышающее 1.
\end{itemize}

В качестве направлений дальнейших исследований следует рассматривать следующие:
\begin{enumerate}
\item повышение эффективности работы с типами данных повышенной точности;
\item применение последних алгоритмических разработок в области параллельного умножения матриц;
\item использование модификаций метода эллипсоидов для решения одномерных задач оптимизации.
\end{enumerate}
