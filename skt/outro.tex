\freesection{Заключение}

В работе представлено методическое пособие, переведенное в формат \LaTeX, а также методические указания к задачам по системам одновременных уравнений.

В работе решены следующие \textbf{задачи}:
\begin{itemize}
\item изучены принципы работы c языком программирования Python;
\item разработано консольное приложение, считывающего данные котировок акций некоторой компании из файла и выводящее среднее значение для цен открытия, закрытия, максимума и минимума;
\item в приложение добавлена поддержка сети;
\item расчетан технический индикатор "индекс денежного потока".
\end{itemize}

По результатам работы можно сделать следующие выводы:
\begin{itemize}
\item в электронном документообороте предпочтительнее использовать формат PDF ввиду его надежности, гибкости и безопасности;
\item разработанные методические указания помогают подойти к изучению СОУ комплексно, предоставляя возможность студентам самостоятельно лучше усваивать материал.
\end{itemize}

Методическое пособие, переведенное в формат PDF, может быть в дальнейшем использовано в Электронном ЮУрГУ -- системе, предназначенной для организации учебного процесса с применением информационных технологий в Южно-Уральском государственном университете.
