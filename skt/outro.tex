\freesection{Заключение}

В работе представлен вариан реализации расчета технического индикатора рынка "индекс денежного потока" на языке программирования Python.

В работе решены следующие \textbf{задачи}:
\begin{itemize}
\item изучены принципы работы c языком программирования Python;
\item разработано консольное приложение, считывающего данные котировок акций некоторой компании из файла и выводящее среднее значение для цен открытия, закрытия, максимума и минимума;
\item в приложение добавлена поддержка сети;
\item расчитан технический индикатор "индекс денежного потока".
\end{itemize}

По результатам работы можно сделать следующие выводы:
\begin{itemize}
\item язык программирования Python очень удобен для быстрой разработки небольших прототипов программ, проверки идей;
\item индекс денежного потока способен подсказать инвестору, каким образом построить свою стратегию в текущих условиях рынка.
\end{itemize}

Приложение может быть использовано для расчета индекса денежного потока для любого исторического периода и любого набора данных.
