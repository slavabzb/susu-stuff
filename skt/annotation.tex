В курсовой работе дается краткое введение в систему технических индикаторов рынка (ТИР), их видов, классификации и исторического развития.  Дается подробное описание алгоритма построения одного из ТИР -- индекса денежного потока для анализа текущих трендов. Приводится программная реализация индекса денежного потока с описанием и комментариями исходного кода.

В приложении приведен полный исходный код приложения для расчета индекса денежного потока.
