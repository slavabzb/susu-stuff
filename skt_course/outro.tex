\freesection{Заключение}

В работе представлена параллельная реализация алгоритма обратного симплекс-метода с использованием субоптимизации и метода наиболее крутого ребра.

В работе удалось реализовать все поставленные \textbf{цели}:
\begin{itemize}
\item изучена общая схема работы алгоритма обратного симплекс-метода;
\item изучены приемы параллельной обработки данных.
\end{itemize}

Решены следующие \textbf{задачи}:
\begin{itemize}
\item разработан класс (тип данных) для решения ЗЛП;
\item разработан решатель, инкапсулирующий работу алгоритма параллельного обратного симплекс-метода;
\item оформлена пояснительная записка, в которой отражены основные этапы работы.
\end{itemize}

В качестве направлений дальнейших исследований можно рассматривать следующие:
\begin{itemize}
\item повышение эффективности работы с типами данных произвольной точности;
\item применение последних алгоритмических разработок в области параллельного умножения матриц;
\item использование различных модификаций симплекс-метода для повышения производительности.
\end{itemize}

Разработанный класс может быть использован для решения семейств взаимосвязанных ЗЛП, для которых другие методы не дают удовлетворительный результат за приемлемое время.
